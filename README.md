# daw-rastreador-onibus
/tEste é o repositório do Sistema de Rastreação dos Ônibus de Belo Horizonte.
O projeto consiste em um website capaz de identificar os ônibus em circulação pela cidade, mostrando suas posições no mapa ao longo de seus trajetos.
É caracteística do projeto possibilitar que o usuário também selecione uma linha especíica dentre as apresentadas. A linha selecionada terá todos os seus pontos mostrados no mapa, bem como terá o ponto mais fisicamente próximo do usuário destacado.
