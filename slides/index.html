Filtração das Linhas

Primeiro para filtrar, foi necéssario usar o bhtrans_bdlinhas.csv para poder conectar os dados da API com a linha do ônibus, uma vez que a API retorna um o código da linha e não o número

A partir do csv foi criado um Map que possui como chave o código da linha e como valor o número da linha, assim permitindo uma conexão direta entre a escolha do usuário e o filtro dos dados.

Por último na hora de desenhar os ônibus, foi utilizado um filtro para exibir apenas a linha desejada.
